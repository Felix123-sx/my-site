## 二、附录式访谈实录

### 受访者R1

基本信息：女，24岁，本科，行政助理，海珠区。

Q1：你觉得自己住的地方发生洪涝的可能性大吗？  
A1：我觉得还是比较大的，尤其是暴雨或者台风来的时候，海珠这边有些路段积水挺明显的。虽然不是每次都会影响到我家，但我会有这个担心。

Q2：你自己或者家里有没有遇到过类似损失？  
A2：我家没有特别严重的损失，但是有一次地下车库进水，邻居的车受影响了。那次之后我会觉得洪涝其实不是离自己很远的事。

Q3：你之前了解过洪涝保险吗？  
A3：以前没有专门了解过，只是听说过家财险。后来在网上看到有人提到气象指数保险，说是达到条件就赔，我才知道还有这种产品。

Q4：你对传统保险印象怎么样？  
A4：我最大的感觉就是条款比较复杂，而且理赔流程有点麻烦。很多年轻人不是完全不接受保险，而是嫌理解成本太高。

Q5：如果有一款洪涝保险，你最看重什么？  
A5：第一是价格不要太高，第二是理赔要快，第三是规则要讲清楚，不然我会犹豫。

Q6：那你会不会考虑购买？  
A6：如果它真的比较透明，而且能看得出有实际保障作用，我会愿意考虑购买。

### 受访者R2

基本信息：男，45岁，高中，小商户，番禺区。

Q1：你觉得你这边下大雨会不会有洪涝风险？  
A1：会，特别是雨大的时候，门口排水跟不上，店里就怕进水。

Q2：以前碰到过吗？  
A2：碰到过一次，店里地面进了水，货虽然没全坏，但清理花了不少时间，人也累。

Q3：你听说过这种洪涝保险吗？  
A3：听说过一点，但说不上了解。以前都是听人讲保险，自己没认真看过。

Q4：你为什么以前没怎么买？  
A4：说白了就是怕麻烦。买的时候讲得好，真出事了又怕赔不下来。

Q5：如果现在有这种保险，什么会让你想买？  
A5：一个是别太贵，再一个就是赔得要快，别让我又交材料又来回跑。

Q6：你现在的态度是什么？  
A6：如果真像你说的那样简单一点，我会考虑，比原来那种保险更愿意试一下。

### 受访者R3

基本信息：女，31岁，硕士，教师，天河区。

Q1：你如何看待广州的洪涝风险？  
A1：我认为广州属于典型的高暴雨暴露城市，洪涝风险具有季节性和区域性叠加特征。尤其在极端天气频发背景下，居民很难完全回避这类风险。

Q2：风险感知会影响你的购买判断吗？  
A2：会。因为一旦我认为这种风险是现实存在且可能造成较大损失的，我就会考虑通过保险做风险分散。

Q3：你了解洪涝保险或气象指数保险吗？  
A3：有一定了解。相较于传统保险，我觉得这类产品的优点在于触发条件相对客观、理赔效率可能更高，但同时也可能存在赔付与实际损失不完全匹配的问题。

Q4：你对传统保险最大的顾虑是什么？  
A4：主要是信息不透明和理赔不确定性。很多消费者的不信任，其实并不是拒绝保险本身，而是担心后续执行落差。

Q5：你最重视哪些产品设计因素？  
A5：我会重点关注赔付触发机制、条款透明度、保费水平和机构公信力。

Q6：你会考虑购买吗？  
A6：如果产品设计清晰、责任边界明确，我会倾向于购买。

### 受访者R4

Q2：家里进过水吗？  
A2：有一次一楼进了水，家具泡坏了点，最后也没人赔，都是自己收拾。

Q3：你以前了解保险吗？  
A3：懂得不多，反正就觉得保险有点绕，合同看不太明白。

Q4：你觉得传统保险哪里不好？  
A4：我最怕的就是到时候说这个不赔那个不赔，最后白花钱。

Q5：要是什么样的保险你觉得能接受？  
A5：简单点，讲明白，多少钱、啥情况赔、啥情况不赔，别绕。

Q6：那你愿意买吗？  
A6：真要是清楚一点，我愿意买，不然心里没底。

### 受访者R5

Q2：你过去有损失经历吗？  
A2：我自己没有大的损失，更多是看到新闻或者朋友圈里别人发积水的情况。

Q3：你对洪涝保险了解吗？  
A3：属于听说过，但是不熟。以前主要关注的是医疗险、意外险，这种灾害保险接触比较少。

Q4：你为什么会观望？  
A4：因为信息还不够。我会担心买了之后到底值不值，保障范围是不是太窄。

Q5：你最在意什么？  
A5：性价比，还有能不能快速知道自己适不适合买。

### 受访者R6

Q1：你对洪涝风险的判断是什么？  
A1：我认为广州洪涝风险总体不低，只是不同区域的体感差异很大。像工作区和居住区的风险暴露就不完全一样。

Q2：你有没有相关经历？  
A2：有过一次车被积水影响的小损失，所以我对这类风险会比较敏感。

A3：了解不算深，但知道它和传统家财险不太一样，重点在于赔付规则更偏客观触发。

Q4：你对传统保险的不满主要是什么？  
A4：理赔效率和沟通成本。很多时候不是赔不赔的问题，而是处理过程太消耗人。

Q5：什么样的产品会打动你？  
A5：保费合理、流程明确、品牌可信，这三点最重要。

Q6：购买意愿如何？  
A6：我属于会观望，但如果产品设计成熟，我是有购买可能的。

Q5：你对产品认知会影响购买吗？  
A5：当然会。认知越清晰，决策阻力越小。

## 四、主轴编码示范

主轴编码阶段，需要将开放编码形成的初始概念进行归类，并上升为二级范畴。

| 二级范畴 | 对应初始概念 |
| --- | --- |
| 风险概率判断 | 感知居住区易涝、风险贴近性增强 |
| 损失经验唤醒 | 有实际损失经历 |
| 风险后果预期 | 担心未来损失、认为洪涝不是离自己很远的事 |
| 产品知晓程度 | 听说过但不了解、对新型保险有基础认知 |
| 产品优势理解 | 认可客观触发赔付、理解理赔更简化 |
| 信息理解障碍 | 认知不足导致观望、看不懂保险条款 |
| 传统理赔不满 | 理赔繁琐、理赔效率低、沟通成本高 |
| 传统条款不满 | 条款复杂、担心执行落差 |
| 价格敏感度 | 别太贵、性价比高才愿意买 |
| 理赔速度偏好 | 赔得快、流程简单 |
| 透明度偏好 | 讲明白保什么、规则透明 |
| 信任背书偏好 | 品牌信誉、真实案例、社区宣传 |
| 保障替代判断 | 自费防灾优先、现有措施替代保险 |
| 购买结果取向 | 愿意购买、观望、暂不购买 |

## 五、选择性编码示范

选择性编码阶段，需要将二级范畴进一步整合为核心范畴，并解释其与购买意愿之间的逻辑关系。

| 核心范畴 | 所含二级范畴 | 与购买意愿的关系 |
| --- | --- | --- |
| 风险感知 | 风险概率判断、损失经验唤醒、风险后果预期 | 风险感知越强，越可能产生风险转移需求 |
| 保险认知 | 产品知晓程度、产品优势理解、信息理解障碍 | 认知越充分，越容易降低决策阻力 |
| 传统保险不满 | 传统理赔不满、传统条款不满 | 对传统保险越不满，越可能转向新型产品 |
| 产品设计偏好 | 价格敏感度、理赔速度偏好、透明度偏好、信任背书偏好 | 产品越契合偏好，购买意愿越高 |
| 情境控制因素 | 保障替代判断 | 对购买意愿形成背景性影响 |
| 结果范畴 | 购买结果取向 | 最终表现为愿意、观望或拒绝 |
